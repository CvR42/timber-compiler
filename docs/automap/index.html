Automap - project summary

Description

The goal of the Automap project is to develop a compiler for high-performance scientific computations. To realize this goal, two important components must be developed: a programming language in which it is convenient to express scientific programs, and a framework that allows programs to be parallelized easily.

The first component, a suitable programming language, is being developed as Spar, a super-set of Java.

The second component, a framework for convenient parallelization, is being developed as the Hazel parallelization engine.

Relations with other projects

Automap provides a significant contribution to the Timber project.
