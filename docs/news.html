News items

13 December 2002: Version 2.0.1 of the Timber compiler is available for downloading. See the version history page for a list of new features.

28 March 2002: Version 2.0 of the Timber compiler is available for downloading. See the version history page for a list of new features.

21 November 2001: Version 1.6 of the Timber compiler is available for downloading. See the version history page for a list of new features.

23 October 2001: Version 1.5 of the Timber compiler is available for downloading. See the version history page for a list of new features.

23 October 2001: A separate web page now compares the language conformance of the Timber compiler with that of traditional compilers. The results may surprise you!

22 March 2001: Version 1.4 of the Timber compiler is available for downloading. See the version history page for a list of new features. Please note that this release requires Kaffe version 1.0.6. This version is also available for downloading.

24 Jan 2001: Version 1.3 of the Timber compiler is available for downloading. See the version history page for a list of new features.
